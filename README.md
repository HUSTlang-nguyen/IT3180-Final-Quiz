# IT3180-Final-Quiz

## User Guide

This document is a collection of multiple-choice questions from 9 chapters of the course **IT3180 - Software Engineering**, compiled and cleaned of OCR errors.

### How to Use This Document

#### 1. Open with Markdown Preview (Recommended)

To be able to **hide/show answers and explanations**, you **must use Markdown Preview**:

**In VS Code:**
- Open the file `quiz_3180_study.md`
- Press `Ctrl+Shift+V` (or `Cmd+Shift+V` on Mac) to open **Markdown Preview**
- Or click the preview icon in the upper right corner of the editor

**On GitHub:**
- Open the file directly on GitHub (github.com)
- Markdown preview will be automatically displayed

#### 2. Answer Hide/Show Mechanism

Each question has the following structure:

```markdown
### Question X
Question content...

- A. Option A
- B. Option B
- ...

<details>
<summary>Show answer and explanation</summary>

**Answer:** [The answer]

**Explanation:** [Detailed explanation]

</details>
```

**How to use:**
- The `<details>` section is **hidden by default** when viewing markdown preview
- Click on **"Show answer and explanation"** to expand/hide the answer section
- This helps you **self-test before viewing the answer**

#### 3. Benefits of Markdown Preview

- **Dynamic Hide/Show:** Easy self-testing, no spoilers
- **Clean Formatting:** Mathematical formulas (KaTeX) display beautifully
- **Images:** Illustrative images display directly
- **Easy to Read:** Optimized layout for studying

#### 4. Content Statistics

- **Total Questions:** 404 questions (after deduplication)
- **9 Chapters:** From software engineering overview to specialized topics
- **Processed:** OCR errors cleaned up, explanations supplemented

## Study Tips

1. **Read the question first:** Try to answer before clicking "Show answer"
2. **Focus on explanations:** The explanation section often contains important knowledge points
